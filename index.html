The Project

Idea and Requirements

Why an Ontology about Sampling?

Everything started with a few questions:

  • Can we use knowledge representation and extraction methods to describe the semantics of sampling within modern music production practices?
  • Could we effectively express which songs — or, indeed, parts of it — have been reused in order to create a new piece of musical artwork?
  • Could we describe the kind of manipulation that the track underwent during the sampling process?
SamO - The Sampling Ontology was created precisely in order to answer these enquiries.

Extending the Music Ontology and beyond

A major starting inspiration for the idea of SamO was the WhoSampled website, with its vast data collection about connections and reuse of songs via covers, remixes and sampling.
For the development of the ontology, the main point of reference was the Music Ontology. By studying the documentation and literature on this ontology, we thought that the best option for developing an ontology about sampling in music would have been to take advantage of some of the already existing classes and predicates from the Music Ontology and to extend it with additional ones.
Nonetheless, the creation of a specific module for dealing with sampling was suggested among the possible future improvements of the Music Ontology in the 2007 paper The Music Ontology.

Requirements and purpose definition (through CQs)

In order to better define the purpose of the ontology, we decided to identify some requirements and relevant questions that an interested user may want to be able to answer by querying a knowledge base. Thus, we created the following competency questions:


Competency Questions

How many samples are present in the song Electric Relaxation?

Which songs are sampled in the song Electric Relaxation?

Which soul songs are used as samples?

How many songs do contain a sample of It feels so good?

Which songs have sampled It feels so good?

Which songs published between 1970 and 1972 have been used as a sample in other songs?

Which songs by Gregory Abbott have been used as a sample?

Which artist is sampled the highest number of times?

Which song contains the highest number of samples?

Which song(s) use(s) samples that have been looped?

Is the sample appearing in Be Happy a direct sample or an interpolation?

How was the sample from Mr Bojangles manipulated in the song Jump Hi?

Who sampled the song Cantaloupe Island?

Who is sampled in the song The words?

All throughout the development, we combined dinamically the user requirements as well as an idea of a possible dataset which would be available for queries and information extraction i.e. we mixed a top-down approach together with a bottom-up one.

The Conceptual Map

We then started to draw our conceptual map as to better distinguish and understand which classes and predicates and datatype and object type properties we were needing. We met several times and discussed in deep how to model some key concepts and especially how to handle crucial events such as the sampling act or the composition act in which we can include the act of creating a new song by reusing somehow some features of a previously sampled track which could or could not have been manipulated, and that at the same time does have a type (whether is direct sample or an exempla of interpolation), and that comprehends some of the musical aspects of a track. We took into prova prova consideration, as already stated, other existing ontologies. We studied them in deep and tried to understand if their semantics of some key musical terms was coherent or somehow alligned with ours, as to reuse but not misuse, let's say, other already existing ontologies, domain specific ones as well as general for the LOD LAM domain (e.g. FRBR core and LRMoo).

CQ #1

How many samples are in the song Electric Relaxation?


SELECT (COUNT(?sample) AS ?count)
WHERE { :Electric_Relaxationsamo:has_sample ?sample }

ANSWER

CQ #2

Which songs are sampled in the song Electric Relaxation?


SELECT ?label
WHERE { ?songsamo:is_sampled_in :Electric_Relaxation ; rdfs:label ?label } SELECT ?label
WHERE { ?songsamo:has_elements ?elements ;
rdfs:label ?label .
?elementssamo:looped_into ?sample .
?samplesamo:used_in :Electric_Relaxation }

ANSWER

CQ #3

Which jazz songs are used as samples?


SELECT ?label
WHERE { ?songsmo:genre :Jazz;
samo:is_sampled_in ?anything ;
rdfs:label ?label }

ANSWER

CQ #4

How many songs contain a sample of It feels so good?


SELECT (COUNT(?song) AS ?count)
WHERE { ?songsamo:samples :It_Feels_So_Good } SELECT (COUNT(DISTINCT ?song) AS ?count)
{ :It_Feels_So_Good samo:has_elements ?element .
?elementsamo:pitch_shifted_into ?sample .
?samplesamo:used_in ?song } SELECT DISTINCT ?song
{ ?songsamo:has_sample ?sample .
?samplesamo:pitch_shift_of ?element .
?elementsamo:part_of :It_Feels_So_Good }

CQ #9

Which artist is sampled the highest number of times?


SELECT ?subject (COUNT(DISTINCT ?song) as ?count)
WHERE {?subjectmo:compiled ?song .
?songsamo:is_sampled_in ?anything }
GROUP BY ?subject
ORDER BY desc(?count)

ANSWER

Which song contains the highest number of samples?
