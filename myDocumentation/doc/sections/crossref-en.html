Analog Samplerc back to ToC or Class ToC

IRI: http://purl.org/ontology/samo/Analog_Sampler

Any sampler that has a physical interface other than a touch screen or keyboard/mouse/screen combination. Intended to cover hardware-based samplers even if they process sound through digital means.
has super-classes
Sampler c
is disjoint with
Digital Sampler c

has super-classes

is disjoint with

Digital Samplerc back to ToC or Class ToC

IRI: http://purl.org/ontology/samo/Digital_Sampler

Any sampler that does not have a physical interface other than a touch screen or keyboard/mouse/screen combination. Intended to cover software-based samplers even if they process sound through analog means.
has super-classes
Sampler c
is disjoint with
Analog Sampler c

Direct Samplec back to ToC or Class ToC

IRI: http://purl.org/ontology/samo/Direct_Sample

Type of sample based on categorization by whosampled.com. Covers all samples created only by a sampler, not any other instrument.
is equivalent to
manipulation_of op some Elements c
has super-classes
Sample c
is disjoint with
Interpolation c

Elementsc back to ToC or Class ToC

IRI: http://purl.org/ontology/samo/Element

Musical elements that constitute a track and are used in the act of sampling, based on the structural logic implied by descriptions of samples on whosampled.com.
is equivalent to
part_of op max 1 track c
has super-classes
signal c
has sub-classes
Bass c, Drums c, Lyrics c, Riff c, Sound Effects c, Voice c
is in domain of
EQed_into op, chopped_into op, clipped_into op, detuned_into op, filtered_into op, looped_into op, manipulated_into op, pitch_shifted_into op, scratched_into op, timestretched_into op, tuned_into op
is in range of
EQed op, chop_of op, clip_of op, detuned op, filtered op, has_elements op, loop_of op, manipulation_of op, pitch_shift_of op, scratch_of op, timestretched op, tuned op

Samplec back to ToC or Class ToC

IRI: http://purl.org/ontology/samo/Sample

Samplingc back to ToC or Class ToC

IRI: http://purl.org/ontology/samo/Sampling
